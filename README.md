# on-the-appearance-of-pi-in-non-circular-domains
my profile work for vwo. regarding the appearance of pi in places where there isn't a clear link to circles. 

## why this subject?
i searched for something that could capture my love for maths without it seeming unbased or narrow. and when talking about pi you can talk about a lot while keeping it strongly based on something everybody undestands

## why latex, isn't it more difficult?
i started with it as having such a large project in LibreOffice or even worse Word would be a bad choice. i expected a lot of difficulty getting to the point where it is intuitive but it happened in just a few hours

### why open source?
because i believe everyone should be able to learn maths for free, and github offers nice tools.
